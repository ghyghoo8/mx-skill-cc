>
> 本文件的价格预测模型是**启发式经验规则，未经任何历史回测验证**，存在已知方法论缺陷，**严禁作为投资决策依据**：
>
> 1. **PE「历史分位」是伪造的**：模型并未取真实历史 PE 序列，而是仅凭*当前* PE 绝对值反推一个假想分位（如 PE≤15 即假设"低估"并给目标 PE ×1.5）。这忽略了行业差异、成长性差异和真实历史区间，**分位结论不可信**。
> 2. **目标价是线性外推**：用净利润同比增速 × 折现系数当作未来 EPS 增长，假设过于简化，对周期股、亏损股、高波动题材股极易失真。
> 3. **三情景倍数（×1.5/×1.0/×0.5）是拍定的**，无统计依据。
> 4. **题材股的价格更多由情绪/资金驱动**，基本面外推模型对其解释力本就有限。
>
> **使用要求**：① 输出目标价时必须同时显示"启发式估算、非回测、仅供参考"字样；② 优先把它当作**相对排序信号**（哪些标的空间相对大）而非**绝对价位预测**；③ 有条件时用 a-stock-data 的同花顺一致预期 EPS（`a_stock_research` `ths_eps_forecast`）替换"净利润同比增速"做更稳健的输入。
>
> 全仓库统一免责：所有分析仅供参考，不构成投资建议。股市有风险，投资需谨慎。

# 中长期价格空间预测模型

## 概述

对 Top5 潜力个股，基于估值修复空间与业绩增长空间，给出 6 个月和 12 个月的目标价区间（乐观/中性/悲观三情景），计算上涨空间百分比并标注预测置信度。

---

## 核心公式

```
目标价（中性） = 当前价 × (1 + 估值修复空间 + 业绩增长空间)
```

### 估值修复空间

```
估值修复空间 = (目标PE - 当前PE) / 当前PE
```

- **目标 PE**: 取行业/个股历史 PE 中位数，默认按 PE 分位规则估算（见下方）
- **当前 PE**: 使用实时 PE(TTM) 数据

### 业绩增长空间

```
业绩增长空间 = 预期EPS增速 × 周期折现系数
```

- **预期 EPS 增速**: 使用净利润同比增速作为代理值（建议改用 a-stock-data 一致预期 EPS，见上方免责第 ③ 条）
- **周期折现系数**: 6 个月 = 0.5，12 个月 = 0.8

---

## PE 分位估算规则

> ⚠️ 见前置免责第 1 条：以下「历史分位」并非真实历史数据，仅由当前 PE 绝对值映射得出，结论不可信，仅作粗排序。

由于实时获取历史 PE 分位数据存在局限性，采用以下简化规则估算 PE 所处历史分位：

| 当前 PE 范围 | 估算历史分位 | 目标 PE 设定 | 含义 |
|:---|:---|:---|:---|
| PE ≤ 15 | 低估（< 20%） | 当前 PE × 1.5 | 严重低估，有较大估值修复空间 |
| 15 < PE ≤ 25 | 偏低（20%-40%） | 当前 PE × 1.3 | 偏低估，有一定修复空间 |
| 25 < PE ≤ 40 | 适中（40%-60%） | 当前 PE × 1.1 | 估值合理，修复空间有限 |
| 40 < PE ≤ 60 | 偏高（60%-80%） | 当前 PE × 0.95 | 估值偏高，存在回落风险 |
| PE > 60 | 高估（> 80%） | 当前 PE × 0.85 | 估值泡沫，修复为负 |
| PE ≤ 0 | 亏损 | 不适用，估值修复空间 = 0 | 公司亏损，PE 无参考价值 |

## 三情景模型

### 情景定义

| 情景 | 估值假设 | 业绩假设 | 说明 |
|:---|:---|:---|:---|
| **乐观** | 估值修复空间 × 1.5 | 业绩增速 × 1.3 | 题材持续发酵 + 业绩超预期 |
| **中性** | 估值修复空间 × 1.0 | 业绩增速 × 1.0 | 题材正常演绎 + 业绩符合预期 |
| **悲观** | 估值修复空间 × 0.5 | 业绩增速 × 0.5 | 题材消退 + 业绩低于预期 |

### 目标价计算

```
乐观目标价 = 当前价 × (1 + 估值修复空间 × 1.5 + 业绩增长空间（乐观）)
中性目标价 = 当前价 × (1 + 估值修复空间 × 1.0 + 业绩增长空间（中性）)
悲观目标价 = 当前价 × (1 + 估值修复空间 × 0.5 + 业绩增长空间（悲观）)
```

其中：
- 业绩增长空间（乐观） = 净利润增速 × 1.3 × 周期折现系数
- 业绩增长空间（中性） = 净利润增速 × 1.0 × 周期折现系数
- 业绩增长空间（悲观） = 净利润增速 × 0.5 × 周期折现系数

### 上涨空间计算

```
上涨空间(%) = (目标价 - 当前价) / 当前价 × 100%
```

> 注：悲观情景下上涨空间可能为负值，表示存在下跌风险。

## 预测置信度

根据数据质量和模型适用性，对每只个股的预测结果标注置信度。

> 注：此处「置信度」衡量的是**输入数据是否齐全/合理**，并不代表模型预测准确——即便置信度为「高」，目标价仍是启发式估算（见前置免责）。

### 置信度评分规则

| 条件 | 置信度分值 |
|:---|:---|
| PE > 0 且 PE ≤ 60（估值数据可用且合理） | +30 |
| 营收增速 > 0（正增长） | +20 |
| 净利润增速 > 0（正增长） | +20 |
| 资金趋势 = 持续流入 或 今日流入 | +15 |
| 题材处于发酵期或高潮期 | +15 |

### 置信度等级

| 总分 | 置信度等级 |
|:---|:---|
| ≥ 70 | **高** |
| 40-69 | **中** |
| < 40 | **低** |

### 置信度说明

- **高**: 估值合理、业绩增长确定、资金支持充分，预测可参考性强
- **中**: 部分维度数据缺失或存在不确定性，预测仅供参考
- **低**: 基本面数据不佳或缺失较多，预测偏差可能较大，需审慎对待

---

## 特殊情况处理

### PE 为负或为零

- 估值修复空间设为 0
- 目标价仅基于业绩增长空间计算
- 置信度自动扣减 30 分

### 净利润增速为负

- 业绩增长空间为负值，会拉低目标价
- 悲观情景下目标价可能大幅低于当前价
- 置信度自动扣减 20 分

### 当前价为零或缺失

- 无法计算目标价，跳过该个股的价格预测
- 输出标注「数据不足，无法预测」

## 操作策略生成

基于价格预测结果，为每只个股生成操作策略。

### 建议仓位

| 条件 | 建议仓位 |
|:---|:---|
| 龙头地位 = 核心龙头 且 置信度 = 高 | 15%-20% |
| 龙头地位 = 核心受益 且 置信度 ≥ 中 | 10%-15% |
| 置信度 = 高 | 10%-15% |
| 置信度 = 中 | 5%-10% |
| 置信度 = 低 | 3%-5%（或观望） |

> 单一题材仓位不超过总仓位 30%。

### 入场时机

| 题材生命周期 | 技术形态 | 入场建议 |
|:---|:---|:---|
| 发酵期 | 量价齐升/放量突破 | 可积极介入 |
| 发酵期 | 缩量上涨 | 等待放量确认后介入 |
| 高潮期 | 任何形态 | 不建议新仓，已持仓者逐步止盈 |
| 分化期 | 核心龙头回调 | 可轻仓博弈反包 |
| 潜伏期/衰退期 | 任何形态 | 观望为主 |

### 止损设置

| 条件 | 止损规则 |
|:---|:---|
| 默认 | 买入价下方 5%-8% |
| 高波动（换手率 > 15%） | 买入价下方 8%-10% |
| 低波动（换手率 < 5%） | 买入价下方 3%-5% |

### 出场信号

- 达到中性目标价时减仓 50%
- 达到乐观目标价时清仓
- 题材进入衰退期时无条件清仓
- 跌破止损位时无条件清仓
- 个股放量滞涨（换手率新高但价格不创新高）
- 所属题材龙头股跌停

## 风险提示

对每只个股，根据预测结果生成个性化风险提示：

| 风险类型 | 触发条件 | 提示文本模板 |
|:---|:---|:---|
| 估值风险 | PE > 60 或 PE < 0 | 当前估值处于历史高位/亏损状态，估值修复空间有限或不适用 |
| 业绩风险 | 净利润增速 < 0 | 公司盈利能力下滑，业绩不达预期风险较高 |
| 题材证伪风险 | 催化剂匹配度得分 < 30 | 公司主营业务与题材关联度有限，题材退潮后回调风险大 |
| 追高风险 | 当日涨停 且 题材处于高潮期 | 短期涨幅已大，追高被套风险较高 |
| 流动性风险 | 总市值 < 30 亿 | 小盘股流动性不足，大资金进出可能造成剧烈波动 |
| 监管风险 | 连续涨停 ≥ 3 天 | 可能触发交易所关注函或停牌核查 |
