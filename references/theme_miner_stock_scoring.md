# 个股五维评分模型与 Top5 筛选

## 概述

在 Top3 热门题材的成分股中，基于「题材核心度 × 技术形态 × 资金面 × 基本面 × 催化剂匹配度」五维评分模型，筛选综合得分最高的 Top5 潜力个股。

---

## 五维评分公式

\[
\text{个股综合得分} = \text{题材核心度} \times 25\% + \text{技术形态} \times 25\% + \text{资金面} \times 20\% + \text{基本面} \times 20\% + \text{催化剂匹配度} \times 10\%
\]

最终得分归一化到 0-100 分。

---

## 维度一：题材核心度（权重 25%）

衡量个股在所属题材中的核心受益程度。

### 评分细则

| 指标 | 条件 | 得分贡献 |
|:---|:---|:---|
| **涨停状态** | 当日涨停 | +35 |
| | 未涨停 | 0 |
| **涨幅排名** | 板块前 3 名 | +25 |
| | 前 5 名 | +15 |
| | 前 10 名 | +8 |
| | 其他 | 0 |
| **市值排名** | 板块内市值前 3 | +15（大市值龙头加成） |
| | 前 10 | +8 |
| | 其他 | 0 |

基础分：15 分。总分限制在 0-100 范围。

### 龙头地位判定

| 条件 | 龙头标签 |
|:---|:---|
| 涨停 且 板块涨幅前 3 | **核心龙头** |
| 板块涨幅前 5 且 市值前 10 | **核心受益** |
| 板块涨幅前 10 | **受益标的** |
| 其他 | **边缘受益** |

---

## 维度二：技术形态（权重 25%）

衡量个股当前的技术走势强度。

### 评分细则

| 指标 | 条件 | 得分贡献 |
|:---|:---|:---|
| **涨幅** | ≥ 9.8%（涨停） | +35 |
| | 5%-9.8% | +20 |
| | 3%-5% | +12 |
| | 0%-3% | +5 |
| | < 0% | -10 |
| **换手率** | ≥ 15% | +25 |
| | 10%-15% | +20 |
| | 5%-10% | +12 |
| | 3%-5% | +5 |
| | < 3% | 0 |
| **量比** | ≥ 3 | +15 |
| | 2-3 | +10 |
| | 1.5-2 | +5 |
| | < 1.5 | 0 |

基础分：15 分。总分限制在 0-100 范围。

### 技术形态标签

| 条件 | 形态标签 |
|:---|:---|
| 涨停 且 换手率 ≥ 10% | **放量涨停突破** |
| 涨幅 ≥ 5% 且 换手率 ≥ 8% | **量价齐升** |
| 涨幅 > 0 且 换手率 < 3% | **缩量上涨** |
| 涨幅 < 0 | **调整中** |
| 其他 | **温和放量** |

---

## 维度三：资金面（权重 20%）

衡量主力资金对个股的参与意愿。

### 数据来源

使用近 5 个交易日的个股资金流向数据（参见 [[theme_miner_data_bridge]] 「个股资金流」→ a-stock-data `a_stock_capital_flow` / `a_stock_signals`）。

### 评分细则

| 指标 | 条件 | 得分贡献 |
|:---|:---|:---|
| **资金趋势** | 持续流入 | +40 |
| | 今日流入 | +20 |
| | 流出 | -10 |
| | 未知 | 0 |
| **累计主力净流入** | ≥ 5000 万元 | +25 |
| | 1000-5000 万元 | +15 |
| | 0-1000 万元 | +8 |
| | < 0 | -5 |
| **最近一日主力净流入** | > 0 | +15 |
| | ≤ 0 | 0 |

基础分：15 分。总分限制在 0-100 范围。

### 资金面标签

| 条件 | 标签 |
|:---|:---|
| 趋势 = 持续流入 | **主力持续加仓** |
| 趋势 = 今日流入 且 最近一日净流入 > 1000万 | **今日大单涌入** |
| 趋势 = 今日流入 | **今日资金流入** |
| 趋势 = 流出 | **资金流出** |
| 趋势 = 未知 | **数据不足** |

---

## 维度四：基本面（权重 20%）

衡量个股的估值合理性与成长性。

### 数据来源

使用个股基本面快照数据（参见 [[theme_miner_data_bridge]] 「个股基本面」→ a-stock-data `a_stock_fundamentals` / 腾讯 `tencent_quote`）。

### 评分细则

| 指标 | 条件 | 得分贡献 |
|:---|:---|:---|
| **营收增速** | ≥ 30% | +25 |
| | 15%-30% | +15 |
| | 0%-15% | +8 |
| | < 0% | -5 |
| **净利润增速** | ≥ 30% | +25 |
| | 15%-30% | +15 |
| | 0%-15% | +8 |
| | < 0% | -5 |
| **PE 合理性** | 0 < PE ≤ 30 | +20 |
| | 30 < PE ≤ 60 | +10 |
| | 60 < PE ≤ 100 | +5 |
| | PE > 100 或 PE < 0 | 0 |
| **ROE** | ≥ 15% | +10 |
| | 10%-15% | +5 |
| | < 10% | 0 |

基础分：10 分。总分限制在 0-100 范围。

---

## 维度五：催化剂匹配度（权重 10%）

衡量个股业务与题材驱动因素的契合程度。

### 评分细则

| 指标 | 条件 | 得分贡献 |
|:---|:---|:---|
| **涨停且属于题材** | 当日涨停 | +40（高度契合） |
| **板块涨幅排名靠前** | 前 5 名 | +25（业务相关度高） |
| **板块涨幅排名中等** | 前 10 名 | +15 |
| **其他** | 排名 10 以外 | +5 |

基础分：20 分。总分限制在 0-100 范围。

> 注：此维度使用个股在板块中的涨幅排名作为契合度的代理指标——涨幅越靠前，说明市场认可其与题材的关联度越高。

---

## Top5 筛选流程

1. 获取 Top3 题材的全部成分股列表
2. 获取涨停股列表，标注哪些成分股当日涨停
3. 对每只成分股计算题材核心度和技术形态得分
4. 对排名靠前的候选股（每个题材取涨幅前 10-15 只），查询资金流向和基本面数据
5. 完成五维评分
6. 全部候选股按综合得分降序排序
7. 取前 5 名作为 Top5 潜力个股
8. 每只个股标注所属题材

### 输出字段

每只 Top5 个股包含：

| 字段 | 说明 |
|:---|:---|
| 排名 | 1-5 |
| 股票代码 | 6 位代码 |
| 股票名称 | 中文简称 |
| 所属题材 | 来自哪个 Top3 题材 |
| 综合得分 | 0-100 |
| 龙头地位 | 核心龙头/核心受益/受益标的/边缘受益 |
| 技术形态 | 放量涨停突破/量价齐升/缩量上涨/温和放量/调整中 |
| 资金趋势 | 持续流入/今日流入/流出/未知 |
| 当前价格 | 元 |
| 涨跌幅 | 百分比 |
| 换手率 | 百分比 |
| PE(TTM) | 倍 |
| 营收增速 | 百分比 |
| 净利润增速 | 百分比 |
| 总市值 | 亿元 |
| 近5日主力净流入 | 万元 |

## 风险标注

对每只入选个股，检查以下风险信号并在输出中标注：

| 风险类型 | 触发条件 | 说明 |
|:---|:---|:---|
| **估值风险** | PE > 100 或 PE < 0 | 估值过高或亏损 |
| **题材证伪风险** | 催化剂匹配度得分 < 30 | 公司业务与题材关联度低 |
| **追高风险** | 涨幅 > 9% 且 所属题材处于高潮期 | 可能处于短期顶部 |
| **流动性风险** | 总市值 < 30 亿 且 换手率 < 3% | 小盘低流动性 |
| **资金撤离风险** | 资金趋势 = 流出 | 主力资金正在撤退 |
