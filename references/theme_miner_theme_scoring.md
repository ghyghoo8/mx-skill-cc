# 题材三维评分模型与 Top3 筛选

## 概述

从全市场概念板块中，基于「驱动强度 × 资金强度 × 可持续性」三维评分模型，筛选出综合得分最高的 Top3 热门题材。每个题材附带驱动逻辑、生命周期阶段、想象空间与操作建议。

---

## 三维评分公式

\[
\text{题材综合得分} = \text{驱动强度得分} \times 40\% + \text{资金强度得分} \times 35\% + \text{可持续性得分} \times 25\%
\]

最终得分归一化到 0-100 分。

---

## 维度一：驱动强度（权重 40%）

衡量题材当前的市场爆发力。

### 评分细则

| 指标 | 条件 | 得分贡献 |
|:---|:---|:---|
| **涨停家数** | ≥ 10 只 | +40 |
| | 5-9 只 | +25 |
| | 3-4 只 | +15 |
| | 1-2 只 | +8 |
| | 0 只 | 0 |
| **板块涨幅** | ≥ 5% | +30 |
| | 3%-5% | +20 |
| | 1%-3% | +10 |
| | 0%-1% | +3 |
| | < 0% | -5 |
| **板块内涨跌比** | 上涨家数/下跌家数 ≥ 5 | +15 |
| | ≥ 3 | +10 |
| | ≥ 2 | +5 |
| | < 1 | -5 |

基础分：15 分。总分限制在 0-100 范围。

### 涨停家数统计方法

遍历涨停股列表，匹配每只涨停股的「涨停原因」标签是否包含该板块名称关键词。匹配到的涨停股计入该板块的涨停家数。

> 数据来源：涨停股列表见 [[theme_miner_data_bridge]] 的「涨停池」补充端点（含 `hybk` 行业归属字段可辅助匹配）。

## 维度二：资金强度（权重 35%）

衡量资金对该题材的参与程度。

### 评分细则

| 指标 | 条件 | 得分贡献 |
|:---|:---|:---|
| **板块成交额** | ≥ 100 亿元 | +30 |
| | 50-100 亿元 | +20 |
| | 20-50 亿元 | +12 |
| | 10-20 亿元 | +5 |
| | < 10 亿元 | 0 |
| **板块换手率** | ≥ 8% | +25 |
| | 5%-8% | +18 |
| | 3%-5% | +10 |
| | 1%-3% | +5 |
| | < 1% | 0 |
| **领涨股涨幅** | ≥ 10%（涨停） | +20 |
| | 5%-10% | +12 |
| | 3%-5% | +5 |
| | < 3% | 0 |

基础分：20 分。总分限制在 0-100 范围。

> 注：板块成交额原始单位为元，评分时换算为亿元（÷ 1e8）。

## 维度三：可持续性（权重 25%）

衡量题材的持续发酵潜力。

### 评分细则

| 指标 | 条件 | 得分贡献 |
|:---|:---|:---|
| **涨停家数** | ≥ 5 只（群体效应） | +25 |
| | 3-4 只 | +15 |
| | 1-2 只 | +8 |
| | 0 只 | 0 |
| **板块内涨跌比** | ≥ 5（普涨） | +25 |
| | ≥ 3 | +15 |
| | ≥ 2 | +8 |
| | < 1 | -10 |
| **板块涨幅适中性** | 2%-5%（健康上涨） | +15 |
| | 5%-8%（偏强但未过热） | +10 |
| | > 8%（可能过热） | +5 |
| | 0%-2% | +5 |
| | < 0% | -10 |
| **连板股数量** | 有连板股出现 | +10（每只连板股 +10，上限 20） |

基础分：20 分。总分限制在 0-100 范围。

## 生命周期判定

每个题材根据当日表现，判定其所处的生命周期阶段。

### 五阶段模型

| 阶段 | 关键标识 | 英文标识 |
|:---|:---|:---|
| **潜伏期** | 板块涨幅 < 2%，涨停数 ≤ 1 | `latent` |
| **发酵期** | 板块涨幅 2%-5%，涨停数 2-5，涨跌比 > 2 | `fermenting` |
| **高潮期** | 板块涨幅 > 5%，涨停数 ≥ 5，普涨 | `climax` |
| **分化期** | 板块涨幅 > 0 但涨跌比 < 2，涨停数有但不多 | `diverging` |
| **衰退期** | 板块涨幅 ≤ 0，涨停数 = 0 | `declining` |

### 判定优先级

按以下顺序依次检查，命中即确定：

1. 板块涨幅 ≤ 0 且 涨停数 = 0 → **衰退期**
2. 板块涨幅 > 5% 且 涨停数 ≥ 5 → **高潮期**
3. 板块涨幅 > 0 且 涨跌比 < 2 且 涨停数 > 0 → **分化期**
4. 涨停数 ≥ 2 且 涨跌比 > 2 → **发酵期**
5. 其他 → **潜伏期**

### 各阶段操作建议

| 阶段 | 操作建议 |
|:---|:---|
| 潜伏期 | 题材尚未启动，可少量布局龙头，等待催化确认 |
| 发酵期 | 最佳介入窗口，积极参与龙头和核心标的 |
| 高潮期 | 谨慎追高，已持有者逐步止盈，关注分化信号 |
| 分化期 | 去弱留强，仅保留最强龙头，回避补涨股 |
| 衰退期 | 回避，不建议新仓介入，等待新题材 |

## 题材级别分类

根据题材的驱动因素性质，分为三个级别：

| 级别 | 判定条件 | 说明 |
|:---|:---|:---|
| **长期战略方向** | 涨停数 ≥ 5 且 板块涨幅 > 3% 且 涨跌比 > 3 | 国家战略级政策驱动，产业空间巨大 |
| **中期产业趋势** | 涨停数 ≥ 3 且 板块涨幅 > 2% | 产业趋势明确，业绩可兑现 |
| **短期事件催化** | 其他情况 | 事件驱动，持续性有限 |

### 各级别想象空间描述模板

- **长期战略方向**: 国家政策重点扶持方向，市场规模达千亿/万亿级，产业替代空间巨大，持续性可达 1-3 年
- **中期产业趋势**: 产业景气度上行，业绩增长确定性较高，受益标的有望走出趋势行情，周期 3-12 个月
- **短期事件催化**: 突发事件/政策催化，市场关注度短期骤升，但持续性需观察，周期 1-4 周

## 题材操作策略生成

根据生命周期和题材级别，生成操作策略：

### 预期周期

| 题材级别 | 预期操作周期 |
|:---|:---|
| 长期战略方向 | 中长线（3-12 个月） |
| 中期产业趋势 | 中线（1-3 个月） |
| 短期事件催化 | 短线（1-2 周） |

### 入场策略

| 生命周期 | 入场建议 |
|:---|:---|
| 潜伏期 | 少量试探，等待放量确认 |
| 发酵期 | 积极布局龙头，量价突破时加仓 |
| 高潮期 | 不建议追高新建仓 |
| 分化期 | 仅龙头回调时可轻仓参与 |
| 衰退期 | 不参与 |

### 出场信号

- 龙头放量滞涨 / 龙头跌停
- 涨停家数从高位急剧萎缩（降幅 > 50%）
- 板块出现大面积跌停
- 题材驱动逻辑被证伪
- 强于大盘的相对强度转弱

## Top3 筛选流程

1. 获取全市场概念板块行情数据
2. 获取涨停股列表，统计每个板块的涨停家数
3. 对每个板块执行三维评分
4. 判定每个板块的生命周期阶段和题材级别
5. 按综合得分降序排序
6. 取前 3 名作为 Top3 热门题材
7. 为每个题材生成驱动逻辑描述、操作策略

### 输出字段

每个 Top3 题材包含：

| 字段 | 说明 |
|:---|:---|
| 排名 | 1-3 |
| 板块代码 | 概念板块代码 |
| 板块名称 | 概念板块名称 |
| 综合得分 | 0-100 |
| 驱动强度得分 | 0-100 |
| 资金强度得分 | 0-100 |
| 可持续性得分 | 0-100 |
| 板块涨幅 | 百分比 |
| 涨停家数 | 只 |
| 生命周期阶段 | 潜伏/发酵/高潮/分化/衰退 |
| 题材级别 | 短期事件催化/中期产业趋势/长期战略方向 |
| 想象空间描述 | 文本 |
| 操作策略 | 含预期周期、入场、出场建议 |
